# Go-Fiber Boilerplate
Golang Rest API built with GORM, Go-Fiber, and a PostgresSQL database. Running in a docker container with Hot Reload.

# File structure
```
database/
  connect.go
  database.go
handlers/
  auth.go
  product.go
middleware/
  json.go
  auth.go
  security.go
model/
  user.go
  product.go
  session.go
router/
  router.go
main.go
```
